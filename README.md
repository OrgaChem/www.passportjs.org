# www.passportjs.org

[passportjs.org](http://passportjs.org/) の Web サイトの非公式日本語訳です。


## 使い方

#### ビルド

このサイトは、[Kerouac](https://github.com/jaredhanson/kerouac) を使って静的な
Web ページを生成している。

    $ npm install
    $ make site
    
#### プレビュー

一度生成したあと、ローカルでサーバーを実行することでプレビューが見られる：

    $ node server.js
    
[http://127.0.0.1:3000/](http://127.0.0.1:3000/) で閲覧可能。

#### 発行

    $ make publish
